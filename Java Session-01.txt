JDK: java development kit 
-----------------------------
Component of JDK:
	-Javac		: java compiler which convert source(.java) to byte code(.class)
	-JVM, JIT	: java virtual machine which manage the memory
				-memory allocation
				-deallocation
				-GC 
			
			: JIT - just in time which is interpreter 
				-convert the byte code to native/machine language/binary code
	-JRE 		: java runtime environment /library

Keywords/Reserved Words
=========================
-package		: is reference of current module/folder
-public			: is access specifier / globally accessiable 
-class			: is wrapper of data member and function/methods 
-void			: is return type / no return 
-static**		: 
			-single memory allocation 
			-can be access/call/execute without object
			
-main			: reserved function / entry point 
-String			: is data type for alpha numeric
-int			: is data type for numeber without decimal 
-double			: is data type for numebr with decimal 
-if			: is conditional statemetn 
-else			: false part of condition 
etc.
